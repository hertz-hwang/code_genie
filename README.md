# 三码优化器 (Three Code Optimizer)

一个基于模拟退火算法的高性能三码键盘布局优化工具，用于优化汉字输入法的编码方案以最小化重码数。

## 📖 项目简介

本项目是一个使用Rust编写的高性能程序，通过模拟退火算法优化汉字输入法的三码编码方案。其核心目标是通过自动化的算法优化，最小化汉字输入法中不同汉字具有相同编码（即重码）的数量，从而提高输入效率。

该工具使用并行处理技术，利用多线程来加速优化过程，并实现了自适应温度控制、扰动机制和增量更新等高级算法特性，以实现更优的优化效果。

## 🚀 功能特点

- **高性能并行优化**: 通过Rayon库实现多线程并行处理
- **模拟退火算法**: 采用先进的模拟退火算法进行优化
- **自适应温度控制**: 根据接受率动态调整温度参数
- **增量更新机制**: 高效计算编码变化对重码的影响
- **强力扰动策略**: 避免陷入局部最优解
- **详细日志输出**: 每个线程都有独立的日志记录
- **结果可视化**: 生成详细的输出文件和总结报告

## 📤 输出文件

程序运行完成后，会在输出目录中生成以下文件：

- `output-字根.txt`: 动态字根键位分配表
- `output-编码.txt`: 汉字的三码编码表
- `log.txt`: 每个线程的详细运行日志
- `总结.txt`: 优化过程的汇总报告

## ⚙️ 配置参数

在 `src/main.rs` 中的 `config` 模块中可以调整以下参数：

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `NUM_THREADS` | 8 | 线程数 |
| `TOTAL_STEPS` | 10,000,000,000 | 每线程迭代步数 |
| `TEMP_START` | 3000.0 | 初始温度 |
| `TEMP_END` | 0.001 | 结束温度 |
| `DECAY_RATE` | 0.999995 | 温度衰减率 |
| `SWAP_PROBABILITY` | 0.6 | 交换操作概率 |
| `PERTURB_STRENGTH` | 0.15 | 扰动强度 |

## 📈 性能优化技术

- **增量计算**: 仅更新受影响的汉字编码，避免全量重新计算
- **并行处理**: 多线程同时进行优化计算
- **自适应参数**: 根据优化进展动态调整算法参数
- **内存优化**: 高效的数据结构和内存管理

## 📚 算法原理

本程序使用模拟退火算法来解决三码编码优化问题：

1. **状态表示**: 每个动态字根分配一个键位(0-25)
2. **目标函数**: 最小化重码数量
3. **邻域操作**: 
   - 交换两个字根的键位
   - 将一个字根移动到新键位
4. **接受准则**: 使用Metropolis准则决定是否接受新状态
5. **温度控制**: 自适应温度衰减策略
